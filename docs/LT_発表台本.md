**意図:** 最初にQRを見せるが「まだ読み取らないで」と伝えることで、逆に興味を引く。

**意図:** 数字のインパクトで「本当に？」と思わせる。

### Slide 5: ライフステージ別比較（45秒）★メインスライド

**表示:** 企画〜運用保守の工程別比較表

**トーク:**

**意図:** 「テスト工程の消滅」は多くのエンジニアにとって衝撃的。

| 質問 | 回答 |

| AI利用料はいくら？ | Claude Code Pro プランで月額約2〜3万円。人件費換算で6〜9倍のコスト削減になる |
| AI が書いたコードの品質は？ | 813テストケース全パス、静的解析エラー0件。セキュリティ対策（XSS、SQLインジェクション）もAIが自動適用 |
